# NVIDIA Model Lists for Multi-LLM Verification

Curated lists of NVIDIA-hosted models optimized for multi-LLM verification tasks.

**Updated:** Daily at 3:00 AM PST via automated benchmarking

## Files

- **`fast-models.json`** - Top 10% fastest conversational models (2-5s response time)
  - Use for: Quick verification, simple questions, routine checks
  - Timeout: 10 seconds
  - Early completion: Returns when 3+ models complete

- **`deep-models.json`** - Slowest 25% most thorough models (15-60s response time)
  - Use for: Deep research, complex analysis, comprehensive comparisons
  - Timeout: 120 seconds
  - Selection: 75-100th percentile (truly thoughtful responses)

- **`benchmarks/`** - [Benchmark history](benchmarks/) (performance tracking over time)
  - [`latest.json`](benchmarks/latest.json) - Most recent benchmark results
  - `YYYY-MM-DD.json` - Daily snapshots (30-day retention)
  - Track model performance trends, availability changes

## Model Selection Criteria

### Conversational Models Only
Both lists exclude:
- Safety/moderation models (llama-guard, nemoguard, etc.)
- Embedding models (bge-*, etc.)
- Reward models (*-reward)
- PII detection models

### Quality Filtering
Models must:
- Respond with >10 words (excludes single-word safety responses)
- Provide conversational output (not just "safe", "on-topic", etc.)
- Successfully complete test prompt about health/longevity

## Benchmark Methodology

Models are tested with:
- **Prompt:** "For a 57-year-old man, what are the top 3 most important daily health habits?"
- **Metrics:** Response time, word count, quality score (structure, completeness)
- **Source:** All available NVIDIA-hosted models via https://integrate.api.nvidia.com/v1
- **Rate limiting:** 40 requests/minute, 5 concurrent

## Usage

### Fast Models (Quick Verification)
```bash
curl -s https://raw.githubusercontent.com/annaxiebot/share/main/fast-models.json | jq '.fast_models[].model'
```

### Deep Models (Thorough Research)
```bash
curl -s https://raw.githubusercontent.com/annaxiebot/share/main/deep-models.json | jq '.deep_models[].model'
```

### In Code
```python
import requests

# Get fast models
fast = requests.get('https://raw.githubusercontent.com/annaxiebot/share/main/fast-models.json').json()
for model in fast['fast_models']:
    print(f"{model['model']} - {model['response_time']}s")

# Get deep models  
deep = requests.get('https://raw.githubusercontent.com/annaxiebot/share/main/deep-models.json').json()
for model in deep['deep_models']:
    print(f"{model['model']} - {model['response_time']}s")
```
